**Modern Scientific Calculator (Python GUI)**

A modern scientific calculator desktop application built with Python using Tkinter.
It provides a clean graphical interface, supports advanced mathematical operations, and includes an interactive calculation history that allows users to reuse previous results.
**Features**
- Basic operations: `+`, `-`, `*`, `/`
- Scientific functions: `sin`, `cos`, `tan`, `log`, `√`, factorial `!`, exponentiation `^`
- Constants: `π` (pi) and `e`
- Clickable calculation history for easy reuse
- Auto-balances parentheses
- Error handling for invalid inputs
- Keyboard support: press Enter to calculate
